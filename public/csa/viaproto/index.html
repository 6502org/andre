CS/A65 VIA prototyping board

This board is a simple prototyping board that you can use to prototype circuits with the VIA. Note that it's untested.

Table of content

  • Board Revisions
  • 1.0A (untested prototype)

    Board revisions

    Version: 1.0A

    Status: untested prototype

    Notes

    This is simply a CS/A board with a VIA and address decoding for it. Empty places for 14/16 pin standard logic ICs (TTL with power on 7/14 resp. 8/16) are included for convenience.
    The board is untested.
