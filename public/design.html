André 's web design

(C) - André Fachat

I have created these pages (under http://www.6502.org/users/andre) with the help of a text editor, an XSLT transformation and some linux shell scripts. For the icons I mainly used the Gimp.

Contrary to most of the content of these pages, the design, stylesheets, icons and javascript is not free for re-use. It is copyrighted by me and re-use requires explicit written permission by me. The exception is the free query javascript library I use, as well as the 2-click social bookmarking code, which comes from heise.de.

The Site has been tested with Firefox 9, IE8, Opera 11 and Google Chrome 16 on Windows XP, as well as Firefox 9, Google Chrome 9 and konqueror 4.3.5 on (SuSE) Linux 11.2, and konqueror 4.7.3 and Firefox 9 on (kubuntu) Linux 11.04.

IE6 is not supported anymore. konquerer 4.3.5 chokes on the JSON javascript object in the social bookmark code, but it is way outdated anyway.
