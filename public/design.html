André 's web design

(C) - André Fachat

I have created these pages (under http://www.6502.org/users/andre) with the help of a text editor, an XSLT transformation and some linux shell scripts. For the icons I mainly used the Gimp.

Contrary to most of the content of these pages, the design, stylesheets, icons and javascript is not free for re-use. It is copyrighted by me and re-use requires explicit written permission by me. The exception is the free query javascript library I use.

The Site has been tested with Firefox 3.6, IE6, IE8, Opera 10 and Google Chrome 5 on Windows XP, as well as Firefox 3.5, Google Chrome 5 and konqueror 4.3.5 on (SuSE) Linux 11.2, and konqueror 4.2.2 and Firefox 3.6.3 on (kubuntu) Linux 10.04.

The only thing I found was that IE6 shows an additional orange bar in the right column, which I happily chose to ignore.
