Software Projects

This section contains projects for PET software.

GeckOS/A65
This is a new operating system for 6502 computers, and has incarnations for the 32k PETs as well as the 8x96 machines. It features multitasking and multithreading, multiple virtual terminals and (on other architectures though) TCP/IP and SCSI support.
LOS-96
This 'operating system' from Commodore is actually an improved BASIC, that allows to utilize the additional memory found in the 8x96 models.
Gecko Serial PET driver
The CS/A Gecko board has a driver that patches up a PET 'Fat40' ROM such that it uses an UART RS232 interface as editor ROM - and not the usual screen area anymore. The page contains a screenshot of a serial terminal with the commodore BASIC startup message :-)
Machine language monitor
Here is the @MON machine language monitor ported to the CRTC PET (40 and 80 cols). Features builtin 6502 assembler, autoscrolling etc

No PETs allowed!
A really cool demo for a 50Hz 4032 PET (with CRTC). There also now is a version without frequency check, that you can try on a 60Hz PET: no_pets_allowed_nohz_check.zip.
