Software projects

   This section contains projects for PET software.

   [2]GeckOS/A65
          This is a new operating system for 6502 computers, and has
          incarnations for the 32k PETs as well as the 8x96 machines. It
          features multitasking and multithreading, multiple virtual terminals
          and (on other architectures though) TCP/IP and SCSI support.

   [3]LOS-96
          This 'operating system' from Commodore is actually an improved BASIC,
          that allows to utilize the additional memory found in the 8x96
          models.
