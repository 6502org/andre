PET index - PET emulators

(C) 1998-2026 André Fachat

PET computers are increasingly hard to get. So, on one side, there are emulators that allow to run PET programs on other hardware.

Also, in the recent couple of years (relative to 2026 where I write this), some new hardware rebuilds, and even mixups with a new hardware just running a PET emulator have come up.

This page tries to describe the different emulators available.

Software Emulators

This section describes emulators implemented in software

VICE
This is the VICE emulator, that not only emulates the PET, but also most other Commodore 8-bit machines (C64, VIC20, C128, CBM-II). In VICE almost all PET computers are emulated (SuperPET 6809 mode is not supported), together with the different IEEE488 disk drives. The VICE emulator runs on different platforms, notably the Windows and Linux/Unix platforms are supprted, but others as well.
Masswerk PET online emulator
This lets you emulate a Commodore PET in your browser! (Note: they also have a nice guide on the PETs in general).
MESS/MAME
MESS emulated a very large number of (even non-Commodore) computers, amongst them also the PET computers. It even says to emulate the SuperPET. The MAME project is said to have inherited it, but the documentation of MAME does not seem to include the PET :-( .
PET Emulator for the C64 by Bob Fairbairn
This is a different type of emulator - it runs on the C64, and emulates the hardware only as far as it is able to catch hardware accesses using BASIC POKE and PEEK commands.
 

Hardware Emulators

This section describes emulators implemented in hardware

The Micro-PET and Ulti-PET
The Micro- and Ulti-PET are complete PET re-builds on a single board. They use all modern components and can be built today without having to scavenge something from an old machine. They use FPGA as video generator, a 65816 CPU for more available memory, and can run up to 13.5 MHz, and have colour video, DAC audio, USB keyboard support. The Ulti-PET adds Dual-SID sound output, joystick ports (compatible with userport Joysticks from the original, Fast-IEC bus support, serial (RS232) interfaces, I2C, SPI interfaces, and even a proper expansion port. More detail on their github page.
MegaPET
A PET core for the Mega65 FPGA retro machine (which started out as a C65 re-build but now runs a lot more different cores, the PET being one of them).
Pet2001-Arty
A recreation of the 2001 on a Xilinux Artix7 FPGA.
MiSTer PET2001 core
A recreation of the 2001 on MiSTer machine, that allows to run a number of different retro machines, the PET being one of them.
MiniPET 40/80
A real PET re-build by Tynemouth software and sold by TFW8b.com (look at their page for kits and other variants)
Commodore PET Mini
This is a PET emulator running on a raspberry pi board, inside a mini PET-like case.
CS/A65
The CS/A65 computer is not a software emulator, but a hardware emulator. It provides a very highly compatible system. In fact it runs a patched 3032 ROM just fine. Only a few IEEE488 lines have been moved around, and the CRTC still needs to be set to 80 columns and not 40 to display 80 columns.
A Commodore PET in an FPGA - Dead project?
This is an amazing emulation - it emulates the complete PET in an FPGA! The emulation is based on a Diligent Nexys3 FPGA evaluation board, implemented in verilog with Xilinx' ISE 13.2. Source code is available! The CPU is not cycle-exact, but it runs Space Invaders and other games just fine. Unfortunately, as of 2026, this project does not seem to be available anymore.
