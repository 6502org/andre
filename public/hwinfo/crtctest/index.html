CRTC test programs and internals

This is a collection of information for the CRTC 6545.

There is NO WARRANTY of any kind attached to the information contained in this collection!!! DO NOT RUN THE PROGRAMS FROM THIS COLLECTION ON YOUR COMPUTER!!!

This includes some test programs for a Commodore PET in the pet directory. The test programs use some timing effects to produce raster effects on the screen. They run on my 8296, but not on yours. I only provide the source as a reference on how to do these timing effects.

The subdirectory internals contains a vague description of the CRTC internals as derived from the results of the test programs.

The CRTC diffs page also summarizes the differences between the different CRTC models.

The operations document contains the CRTC description of the PET index.
