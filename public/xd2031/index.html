XD2031 - AtMega(tm) firmware for virtual CBM drives

(C) 2012-2013 André Fachat

The XD2031 project is an AtMega(tm) firmware for devices like the petSD or xs1541, to connect Commodore devices to the PC, and use the PC as disk drive via an USB connection.

This software enables to program an AVR AtMega device to play the role of a disk drive to a Commodore PET computer with IEEE488 interface.

XD-2031 consists of two main parts, an AtMega firmware, and a PC server. Both communicate via a serial line.

Currently this hardware is supported:

  • XS-1541 Communication via serial line over USB
  • petSD Communication via serial line over USB

For more information see the README files in the repository.
