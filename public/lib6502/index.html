lib6502 standard

(C) 1997-2010 André Fachat

The lib6502 standard library for 6502 systems is far from being complete. The current version is lib6502.html, but older versions can be obtained from below.

The standard is now heading not only for source compatibility, but also for binary compatibility. It is still possible to claim source compatibility, but when using the values (for error codes, function codes,...) (- not yet all -) given in this lib, the files should be binary compatible.
